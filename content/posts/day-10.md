date: "2026-04-20"

Today was a big shift in how I think about design. I learned that UI kits are not just about making things look good, they help me build faster by using proven components and patterns. Instead of designing everything from scratch, I focused on assembling screens using existing elements like tabs, cards, and forms. This made the process feel more structured and closer to how real products are built.

I also started thinking more about the system behind the app, like how user input turns into data and then into useful output. That helped me design with more intention instead of just focusing on visuals. I made solid progress by defining my main screens and beginning to lay out my fitness app using a UI kit.

I still need to improve on connecting everything into a smooth user journey, but I feel like I’m building on a stronger foundation now.

I’m grateful for the clarity I gained today and for learning a process that helps me move faster without overthinking every detail.
