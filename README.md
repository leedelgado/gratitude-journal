# Gratitude Journal

A personal AI learning journey, documented using a beautiful, minimalist markdown-based static site built with Next.js App Router.

## Getting Started

First, install dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Adding new entries
To write a new gratitude journal entry, simply create a new markdown file in the `content/posts/` directory and include the metadata at the top:

```markdown
---
title: "Your Title Here"
date: "YYYY-MM-DD"
---

Your content here.
```
